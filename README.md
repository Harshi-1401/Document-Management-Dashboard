# Document Management Dashboard

A full-stack web app to upload, manage, and track PDF documents with real-time notifications.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS + Axios
- Backend: Node.js + Express + Multer + Socket.IO
- Database: MySQL (mysql2)

## Setup

### Backend
```bash
cd backend
npm install
# Update .env with your MySQL credentials
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
